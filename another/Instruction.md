## 📜 Требования:

- DotNet для сборки программ (Если вылезают ошибки VCRUNTIME140.dll, Из архива требований запустите VC\_redist.x64.exe (или скачать и запустить с браузера: [*https://aka.ms/vc14/vc\_redist.x64.exe*](https://aka.ms/vc14/vc_redist.x64.exe)))
- Установить WebView2
- Была установлена Windows 11
- Redist для работы IDE

## 📥 Инструкция по установке
0. `Подготовка`: Проверьте наличий обновлений `Windows, Microsoft Edge и Microsoft Store`.
1. Скачайте последнюю версию `Install_vucapp.exe` из раздела [Releases](https://github.com/VentraKat/VUCAPP/releases).
2. Запустите файлы `dotnet-*.exe` и `redist.exe` для установки необходимых системных компонентов. Из архива требований
3. Запустите файл `Install_VUCAPP.exe`.
4. В открывшемся окне нажмите **"Установить"** -> дождитесь окончания установки -> закройте окно установщика.
5. Запустите программу по пути: `C:\vucapp\launcher.exe` *(если брандмауэр Windows запросит разрешение на доступ к сети — нажмите "Да")*.
6. **Установка пакетов:** Перейдите в `C:\vucapp\nugets\` и запустите скрипт `InstallNugets.bat`. Дождитесь завершения процесса и нажмите Enter.
