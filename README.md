## 💡 Что такое VUCAPP?
VUCAPP — это бесплатная инновационная IDE с визуальным интерфейсом, решающая проблему сложного входа в разработку. Она позволяет легко создавать полноценные `.exe` программы под Windows 11 с современным дизайном **WinUI 3**. 

Идеально подходит для новичков и "ленивых" разработчиков: вы собираете красивый интерфейс мышкой (Drag-and-drop), а VUCAPP автоматически генерирует чистый код на **C#**.

## ✨ Ключевые возможности
* **Визуальное программирование:** Drag-and-drop конструктор интерфейсов.
* **Дизайн WinUI 3:** Нативные элементы управления Windows 11 с поддержкой Mica и темной темы.
* **Генерация кода:** Автоматическое создание читаемого C# кода из визуального представления.
* **Встроенный NuGet:** Удобный менеджер пакетов для расширения функционала программ.
* **Шаблоны проектов:** Готовые заготовки (например, браузер или текстовый редактор) для быстрого старта.
* **Компиляция в 1 клик:** Сборка проекта в независимый `.exe` файл через встроенные скрипты и .NET SDK.

> [!NOTE]
> **Разработка для ColdOS (ColdAPP):** Поддержка и разработка данного направления на данный момент отложена.
>
> Но работа функция: У вас будет выбор:
> - Создать приложение Windows
> - Создать приложение ColdOS

## 📥 Инструкция по установке
0. `Подготовка`: Проверьте наличий обновлений `Windows, Microsoft Edge и Microsoft Store`.
1. Скачайте последнюю версию архива `VUCAPP.zip` из раздела [Releases](https://github.com/VentraKat/VUCAPP/releases).
2. Распакуйте скачанный архив в удобное место.
3. Откройте папку `fixes` и запустите файлы `dotnet-*.exe` и `redist.exe` для установки необходимых системных компонентов.
4. Запустите файл `Install_VUCAPP.exe`.
5. В открывшемся окне нажмите **"Установить"** -> дождитесь окончания установки -> закройте окно установщика.
6. Запустите программу по пути: `C:\vucapp\launcher.exe` *(если брандмауэр Windows запросит разрешение на доступ к сети — нажмите "Да")*.
7. **Установка пакетов:** Перейдите в `C:\vucapp\nugets\` и запустите скрипт `InstallNugets.bat`. Дождитесь завершения процесса и нажмите Enter.
