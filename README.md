## 💡 Что такое VUCAPP?
VUCAPP — это IDE с визуальным интерфейсом. Она позволяет легко создавать полноценные `.exe` программы под Windows 11 с современным дизайном **WinUI 3**. 

Идеально подходит для новичков, но и для обычных разработчиков: вы собираете красивый интерфейс мышкой, а VUCAPP автоматически генерирует чистый код на **C#**.

## ✨ Ключевые возможности
* **Визуальное программирование:** Drag-and-drop конструктор интерфейсов.
* **Дизайн WinUI 3:** Нативные элементы управления Windows 11 с поддержкой Mica и темной темы.
* **Генерация кода:** Автоматическое создание читаемого C# кода из визуального представления.
* **Встроенный NuGet:** Удобный менеджер пакетов для расширения функционала программ.
* **Демонстрационные проекты:** Готовые заготовки (например, браузер или текстовый редактор) для быстрого старта.
* **Компиляция в 1 клик:** Сборка проекта в независимый `.exe` файл через встроенные скрипты и .NET SDK.

> [!NOTE]
> **Разработка для ColdOS (ColdAPP):** Поддержка и разработка данного направления на данный момент отложена.
>
> Но работа функция: У вас будет выбор:
> - Создать приложение Windows
> - Создать приложение ColdOS
