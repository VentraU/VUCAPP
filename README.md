# VUCAPP

- 👋 Это среда разработки приложений с визуальным интерфейсом, созданная на языке **HTML, JS**.
- ✨ Поддерживает создание приложение для Windows 11 и ColdAPP
- ❤️ Ориентирован на ленивых разработчиков и новичков, которым нужен простой, визуальный способ создания приложений под Windows и ColdOS.

## ⚙️ Возможности для Windows приложений:

- 🎨 Визуальный редактор интерфейса (Конструктор дизайна)
- 📦 Добавление и настройка компонентов: кнопки, чекбоксы, поля ввода и др.
- ⚙️ Автоматическая генерация исходного кода на C#.
- 🚀 Возможность сборки и компиляции проекта в `.exe`.
- 📜 Подключение кода, к компоненту.

> ## 🔧 Возможности для ColdOS приложений (ColdAPP) (в процессе разработки):

> - 🎨 Визуальный редактор интерфейса (Конструктор дизайна)
> - 📦 Добавление и настройка компонентов: кнопки, чекбоксы, поля ввода и др.
> - ⚙️ Автоматическая генерация исходного кода в директной структуре.
> - 🚀 Возможность сборки и компиляции проекта в `.coldapp`
> - 📜 Подключение JS кода, к компонента

## 📥 Как запустить (На данный момент)
1. Скачайте приложение установки из **Релизов**:
```
VUCAPP.zip
```
2. Распакуйте архив.
4. Убедитесь, что установлен **.NET SDK** и **редист**, если нет, запустите с папки fixes.
5. Запустите `Install_VUCAPP.exe`.
6. Нажмите "Установить".
7. Закройте установщик.
8. После запуска подтвердите запуск через общий доступ, нажав ДА.
9. Перед компиляции вашей программы, запустите `C:\vucapp\nugets\InstallNugets.bat` введите на (данный момент) `8.0.24`
