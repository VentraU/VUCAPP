# ⚒️ VUCAPP — VentraUniverseCreateAPP
**VUCAPP** — это современная альтернатива **PHP Devel Studio**, разработанная на языке **HTML, JS**.
Проект ориентирован на разработчиков и новичков, которым нужен простой, визуальный способ создания приложений под Windows и ColdOS.

## 🔧 Возможности (🪟 Windows) (в процессе разработки):

- 🎨 Визуальный редактор интерфейса (Конструктор дизайна)
- 📦 Добавление и настройка компонентов: кнопки, чекбоксы, поля ввода и др.
- ⚙️ Автоматическая генерация исходного кода на C#.
- 🚀 Возможность сборки и компиляции проекта в `.exe`.
- 📜 Подключение С# код, к компонента

## 📥 Как запустить (На данный момент)
1. Скачайте файл установки из **Релизов**:
```
VUCAPPsetup.zip
```
2. Распакуйте архив.
3. Переместите папку `vucapp` в диск `C:\`.
4. Убедитесь, что установлен **.NET SDK** и **редист**, если нет, запустите с папки fixes.
5. Запустите `C:\vucapp\launcher.exe`.
6. После запуска подтвердите запуск через общий доступ, нажав ДА.
7. Перед компиляции вашей программы, запустите `C:\vucapp\nugets\InstallNugets.bat` введите на (данный момент) `8.0.24`
